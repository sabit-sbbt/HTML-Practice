• Bootstrap-4 is Mobile-First

Bootstrap 4 is desgined to be responsive with mobile. Mobile-First styles are part of core-framework.

To ensure proper rendering and touch zoming, it is needed to add <meta> tag inside the <head> element with below structure↓

<meta name="viewport" content="width=device-width, initial-scale=1">

Bootstrap-4 Container


Bootstrap requires a containing element to wrap site contents. Containers are used to pad content inside them, there are two different continer classes available:-

  1. Fixed-Width Container: The .container provides a responsive fixed-width container. Its sizes are pre-fixed
  2. Full-Width Container: The .container-fluid provides a full-width container, spanning the entire wwidth of the viewport
